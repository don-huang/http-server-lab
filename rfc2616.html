RFC 2616 - Hypertext Transfer Protocol -- HTTP/1.1

This document specifies version 1.1 of the Hypertext Transfer Protocol (HTTP), a generic and stateless protocol used by the World Wide Web. This version improves efficiency and flexibility compared to HTTP/1.0.

Example Request

GET /index.html HTTP/1.1
Host: www.example.com
Connection: close
User-Agent: ExampleBrowser/1.0

Example Response

HTTP/1.1 200 OK
Date: Sat, 17 May 2025 08:00:00 GMT
Server: ExampleServer/1.0
Content-Type: text/html; charset=UTF-8
Content-Length: 1234


  
    
Hello, HTTP!

Status Codes

  • 200 OK – The request has succeeded.
  • 301 Moved Permanently – The resource has been moved to a new URL.
  • 302 Found – The resource is temporarily available at a different URI.
  • 403 Forbidden – Access to the resource is denied.
  • 404 Not Found – The resource could not be found.

Connection Handling

HTTP/1.1 introduces persistent connections, meaning that multiple requests can be sent over the same TCP connection unless explicitly closed.
